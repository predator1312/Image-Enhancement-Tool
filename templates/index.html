About This Program

This application enhances image quality and detail using advanced wavelet transformation techniques and adaptive contrast improvement.

Technologies Used

  • DWT (Discrete Wavelet Transform): Decomposes the image into different frequency components.
  • SWT (Stationary Wavelet Transform): Provides translation-invariant wavelet decomposition.
  • Cubic Spline Interpolation: High-quality image resizing preserving edges.

Enhancement Techniques

  • Thresholding: Applies hard or soft thresholding to reduce noise while preserving features.
  • CLAHE: Enhances local contrast while preventing noise amplification.
  • Combined Approach: Uses wavelets with interpolation for optimal results.

Implementation

  • Backend: Python with Flask web framework
  • Libraries: NumPy, OpenCV, PyWavelets, SciPy
  • Frontend: HTML, CSS, JavaScript
